Petinder

Aplicativo web para matching entre pets!

Saiba mais...

O PeTinder é uma plataforma web de matchmaking para pets, desenvolvida com Python, Flask e SQLite. Funciona como o Tinder: o tutor cadastra seu animal com fotos e filtros como raça, gênero e pedigree, navega por candidatos da mesma cidade e espécie, e quando dois tutores dão like um no pet do outro ocorre um match — ambos são notificados com o contato do outro tutor. O projeto segue arquitetura MVC em camadas, com sistema de planos Free e Premium, painel administrativo, notificações em tempo real e autenticação segura com hash bcrypt. Tecnologias: Python, Flask, SQLite, SQLAlchemy, Bootstrap 5, Jinja2.

Biblioteca 2.0

Sistema de Gerenciamento de Biblioteca!

Sistema web de gerenciamento de biblioteca desenvolvido com Flask e MySQL. A aplicação segue a arquitetura em três camadas e oferece tanto uma API REST quanto uma interface visual para cadastro e controle de usuários, livros e empréstimos.
